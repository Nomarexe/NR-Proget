# NR Project – Dinamico e Statico

NR Project è una piattaforma web multimediale completa, disponibile in due versioni distinte:

- **NR Project Dynamic Python Server**: versione dinamica con backend Flask e API per contenuti multimediali
- **NR Project Static**: versione statica HTML/CSS/JS, pronta per il deploy su hosting statici come GitHub Pages o Vercel

Entrambe le versioni offrono un'interfaccia coerente, responsive e intuitiva, con supporto per immagini, video, audio e documenti.

## Versioni disponibili

| Versione  | Descrizione                                                                 |
|-----------|------------------------------------------------------------------------------|
| Dinamica  | Server Python Flask con caricamento dinamico e API dedicate                 |
| Statica   | Sito statico HTML/CSS/JS, navigabile senza backend                          |

## Releases

Le versioni ufficiali sono disponibili nella sezione [Releases del repository](https://github.com/Nomarexe/NR-Proget/releases).

## Requisiti (solo per la versione dinamica)

- Python 3.7 o superiore
- pip
- bash o zsh (per lo script di setup)

## Avvio versione dinamica

```bash
bash setup_instructions.sh
python3 app.py
```
Accedi al sito su http://127.0.0.1:8080
